# PerfumeInsightLab

Perfume analytics project, from raw data to scent-based insights.  
**EDA** (Exploratory Data Analysis) on perfume datasets to uncover patterns, trends and insights.

---

## Overview  
**PerfumeInsightLab** aims to :
- Clean and structure raw perfume datasets  
- Explore key features and relationships (brands, accords, notes, ratings, yearsâ€¦)  
- Visualize trends and distributions  
- Generate insights on fragrance characteristics, composition, popularity, market trends 
- Prepare the data for SQL analysis and potential machine learning extensions  

## Tech Stack  
- **Python 3**  
- **Pandas / NumPy** (data cleaning & manipulation)  
- **Matplotlib / Seaborn / Plotly** (visualizations)  
- **WordCloud** (text-based visualization of notes)  
- **SQL**/**SSMS** (queries and data exploration in databases)
